## Overview
 
`quantum_toolkit` is an open-source, educational Python simulation tool for exploring quantum mechanics concepts interactively. It allows students to write, solve, and visualize the Schrödinger equation (time-dependent and time-independent) for various 1D potential shapes, making abstract quantum formalism tangible through code and live plots.
 
The project emphasizes **pedagogical transparency** — every computation is explicit and readable, unlike closed-source tools such as PhET simulations.
